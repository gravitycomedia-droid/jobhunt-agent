Line-icon primitive — a curated Lucide-style glyph set, stroke-based, inherits `currentColor`; use for all in-app iconography.

```jsx
<Icon name="briefcase" size={22} />
<Icon name="check" color="var(--success-fill)" />
```

Names cover the five nav destinations (home, briefcase, target, columns, user), job/meta glyphs (mapPin, building, externalLink, bookmark), status glyphs (check, x, minus, alertTriangle, info) and chrome (search, bell, chevronDown, plus, clock, bot, upload, fileText). Size + strokeWidth are tunable; default 20 / 2.
